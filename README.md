# 🎬 Netflix Tech

Projeto desenvolvido durante a **Imersão Front-End da Alura**, com o objetivo de recriar a interface da Netflix com uma proposta voltada ao mundo da tecnologia.

## 🚀 Demonstração

🔗 Acesse o projeto:
https://joaodevguilherme.github.io/IMERSAO_ALURA_FRONT_END/

---

## 📌 Sobre o projeto

O **Netflix Tech** é uma aplicação front-end que simula a experiência de seleção de perfis da Netflix, adaptada para áreas da tecnologia como:

* 👨‍💻 Frontend
* ⚙️ Backend
* 🔄 Full Stack
* ☁️ DevOps

Ao selecionar um perfil, o usuário é direcionado para um catálogo específico.

---

## 🛠️ Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript
